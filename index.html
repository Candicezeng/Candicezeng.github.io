sys.argv的用法

sys.argv[]的作用是获取命令行的参数，sys.argv[0]代表的是文件本身的路径。从sys.argv[1]开始是文件执行的参数。废话不多说，先上例子。
使用sys.argv的例子：
在此，我建立一个test.py的Python文件，内容如下：

[Python]
import sys
print ‘Number of arguments:’, len(sys.argv), ‘arguments.’
print ‘Argument List:’,str(sys.argv)

现在我们在terminal中运行test.py.在terminal中输入：

python test.py arg1 arg2 arg3

这样我们会得到：

Number of arguments:4 arguments.
Argument List: [‘test.py’,’arg1’, ‘arg2’, ‘arg3’]

这里的四个arguments分别是：test.py, arg1,arg2,arg3。其中test.py为sys.argv[0],arg1为sys.argv[1]，以此类推。那么这样的参数有什么作用呢？在接下来的getopt中会体现出其作用。
