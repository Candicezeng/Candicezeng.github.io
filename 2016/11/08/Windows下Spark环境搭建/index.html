Windows下Spark环境搭建

发表于 2016-11-08   |   分类于 Spark   |     |  

最近在听Berkeley的介绍Apache Spark课程，所以就在本地搭建了pySpark的环境，在此记录在Windows下Spark环境搭建过程。
Berkeley CS105x Introduction to Apache Spark 传送门：CS105x

我搭建的环境为：windows10+spark2.0.1+python 2.7。

安装JDK

安装spark的第一步就是安装jdk，spark依赖于jdk执行。jdk可在oracle官网上下载，下载传送门JDK

在这里我选择的Windows x64版本，你可根据自己的需求下载。下载完成后选择安装路径安装就好。在此，我直接选择默认路径安装，即C:\Program Files\Java。安装好后就需要设置环境变量了。依次打开system–>Advanced system setting-
->Advanced->Environment Variable

添加两个环境变量：
JAVA_HOME：
CLASSPATH：
然后打开PATH,添加

%JAVA_HOME%\bin

然后打开cmd窗口，执行：

java -version

出现java的版本信息则安装成功了。

安装Spark

去Spark官网下载spark，传送门SPARK
在此我选择的是spark2.0.1，其对应Hadoop的版本为Hadoop2.7版本及以后。

下载完成后解压缩到你想要放的位置，我直接将Spark放在了C盘下。然后添加Spark环境变量。

在path中添加：

%SPARK_HOME%\bin
%SPARK_HOME%\sbin

安装hadoop

下载安装hadoop，传送门Hadoop
在此我选择的是Hadoop 2.7.1，以对应我的spark版本。下载完成后解压缩，添加环境变量HADOOP_HOME。如下：

在path中添加：

%HADOOP_HOME%\bin

在此为了避免出现问题，我还安装了scala，同时添加了SCALA_HOME环境变量，并在path中添加了%SCALA_HOME%\bin，在此就不赘述了。

pyspark设置

在python环境运行pip install py4j.

安装完成！在pycharm中开始编程吧！
